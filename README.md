# Foreign-Born-Worker-Influence-on-Native-Employment-in-Nebraska-and-Beyond

1. Project Title

Foreign-Born Worker Influence on Native Employment

2. Project Overview

This project analyzes employment patterns between native-born and foreign-born workers in the United States, with a focus on Nebraska. The goal is to examine whether foreign-born workers affect employment opportunities for native-born workers and explore differences in labor participation.

3. Dataset

The project uses data from the American Community Survey (ACS) and U.S. Census demographic datasets. The data includes variables such as population, employment status, and worker classifications.

Source: U.S. Census Bureau and Kaggle.

4. Methods

The analysis includes:

Data cleaning and preparation

Data merging from multiple sources

Exploratory data analysis (EDA)

Statistical analysis and visualization

5. Tools Used

Python

Pandas

Matplotlib / Seaborn

Jupyter Notebook

6. Key Insight

The analysis compares employment trends between native-born and foreign-born workers and provides insight into labor participation patterns in Nebraska and across the United States.
